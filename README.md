# Keylogger básico con Python 

## Descripción
Este proyecto implementa un programa sencillo que detecta teclas presionadas utilizando la librería keyboard y guarda las letras en un archivo de texto (log.txt).

El objetivo es comprender cómo funcionan este tipo de herramientas en entornos controlados con fines educativos.

## Requisitos
- Python 3.10 o superior
- Librería keyboard

## Instalación

Instalar dependencias:
pip install -r requirements.txt

## Ejecución

Ejecutar el programa:
python main.py

## Funcionamiento

- El programa detecta las teclas presionadas
- Guarda únicamente letras (a-z) y espacios en log.txt
- Se ejecuta en tiempo real
- Para detenerlo se presiona la tecla ESC

## Ejemplo de uso

1. Ejecutar el programa
2. Escribir texto en el teclado
3. Presionar ESC para finalizar
4. Revisar el archivo log.txt

## Ejemplo de salida

Contenido de log.txt:

hola mundo
